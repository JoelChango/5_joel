    Intro

    La construccion de una pagina web es un arte que combina creactividad con tecnologia .Para que esta creacion sea efectica y accesible,necesita una herramienta fundamental: el HTML.Desde su creacionen 1991.el HTML ha sido el pilar sobre el que se construye una pagina web A traves de etiquetas estructuras y atributos,permite a los desarrolladores dar forma a la informacion que vemos y consumimos diariamente emn internet.En este Capitulo exploraremos los conceptos basicos de HTML desglosando su estructura y sintaxis para comprender como funciona

    Desarrollo

    HTML funciona mediante el uso de funciones o tags ,que indican al navegador como debe mostrarse el contenido.Estas etiquetas estan delimitadas por corchetes angulares , y cada etiqueta tiene su funcion especifica

    Final

    El HTML es mas que un simple lenguaje marcado; es el cimiento sobre el cual se construye la web moderna.Aunque su sintaxis es relativamente sencilla es ,las posibilidades que ofrece son vastas cuando se combina con CCS o Java Script.Hemos aprendido las bases de HTML desde las etiquetas mas fundamentales hasta la importancia de los atributos y las nuevas caracteristicas de HTML5. Comprender estos fundamentos es esencial para cualquier desarrollador web ,ya que constituye el primer paso en el camino hacia la creacion de experiencias digitales y y funcionales
